# Calculator
用java写的一个简易的安卓计算器

目前拥有的功能：

  计算功能：
  
    加减乘除 括号 三角函数 log 开根号 次放 阶乘 百分比
    
  软件功能：
  修复一些bug
    清零 删除 等于 点击等于再点击删除后会自动在之前输入的内容上操作而非在运算结果上操作 
    随着输入显示结果 可以在输入内容的任意位置再修改或输入
    
  UI：
  
    仿照Google Calculator
